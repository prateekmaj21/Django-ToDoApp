# How to use?

1. Download the files. 
2. Have Django 3 installed.
3. Navigate to the "todoapp" directory. 

4. Check Django Version
```
python -m django --version

```
5. Make sure Directory is Ok.

```
cd todoapp
```
6. Run the project. 
```
python manage.py runserver
```
